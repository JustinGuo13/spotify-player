## Spotify Player for Premium Spotify

https://pguo-spotify.vercel.app/

It's a nice Spotify Player app developed by me in order to practice the use of Next.js, TailwindCSS, and NextAuth.js.

Used:

-   Next.js
-   TailwindCSS
-   NextAuth.js
-   Spotify API
-   Recoil

Application allows:

-   To play/pause music
-   To change playlists
-   To change volume
-   To skip tracks
